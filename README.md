# Image-Video-Object-Detection
An advanced app for detecting objects in images and videos with tensorflow-models. <br/>
Create, analyze, and explore real-time object detection. [Check it here](https://shivanimakvana.github.io/Image-Video-Object-Detection/).

## Paths
- / (for live video feed detection)
- /image (to upload image for image based object detection)

## Tensorflow models 
- coco-ssd for Live feed detection
- mobilenet for image recognition
